# Cosmos VLA Drone

A research project for building a Cosmos-enhanced vision-language-action drone agent in Isaac Sim with spatial, semantic, and episodic memory.

## Goal

This project extends a previous VLA drone prototype into a high-fidelity embodied AI research system. The system will use Isaac Sim for simulation, Cosmos for world understanding and synthetic data support, and memory modules for long-horizon exploration.

## Research Direction

- Vision-language-action drone control
- Isaac Sim high-fidelity simulation
- Cosmos-enhanced world understanding
- Spatial, semantic, and episodic memory
- Policy learning and post-training
- Synthetic data generation for embodied agents
